# Evaluator Matrix

| Subsystem | Evaluator | Mandatory/Optional | Cost | Source-of-truth artifact | Command |
| --- | --- | --- | --- | --- | --- |
| Cross-repo behavior | Jest unit/integration coverage gate | Mandatory | Medium | Terminal output + coverage summary generated by Jest | `npm test` |
| Style and static hygiene | ESLint + Prettier formatting gate | Mandatory | Medium | `reports/lint/lint.txt` | `npm run lint` |
| Static blog generation | Build pipeline smoke | Mandatory for build changes | Medium | Generated assets under `public/` + command output | `npm run build` |
| Dendrite static packaging | Copy pipeline verification | Optional unless Dendrite asset paths touched | Low | Copied artifacts under infra/public targets + command output | `npm run build:dendritestories-co-nz` |
| Cloud packaging | Deploy package assembly check | Optional unless `src/cloud`/cloud build paths touched | Low-Medium | Assembled deploy files under infra/cloud targets + command output | `npm run build:cloud` |
| Local writer runtime | Server boot smoke + endpoint sanity | Optional unless local writer paths touched | Low | Local server logs and HTTP probe output | `npm run start:writer` |
| End-to-end user journey | Playwright cloud E2E | Optional for local loops, Mandatory in CI release gates | High | Playwright report artifacts in CI / `playwright-report/` | `npm run test:e2e` |
| Dependency architecture | Duplication and coupling trend check | Optional | Medium | `reports/duplication` and dependency analysis output | `npm run duplication` |
| Mutation confidence | Behavioral robustness under mutation | Optional scheduled check | High | `reports/mutation/text/mutation.txt` | `npm run stryker` |
