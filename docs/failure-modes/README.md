# Failure Modes Catalog

This directory tracks repeatable failure surfaces and their operating playbooks.
Each file should convert ad-hoc debugging into a reusable loop artifact.

## How to use

1. Identify the nearest matching surface.
2. Confirm symptom and detection signal.
3. Run the documented fix path.
4. If the issue repeats, strengthen the prevention harness.

- `ci-cd.md`
- `local-server.md`
- `infra.md`
